## How it works

<Steps>
  <Step title="Browse with HAR recording">
    Launch a Playwright browser with stealth mode. Navigate and interact with the target website while all network traffic is captured to an HTTP Archive (HAR) file.
  </Step>
  <Step title="AI analyzes traffic">
    Claude 4.6 reads the HAR file, identifies API endpoints, authentication patterns, request/response schemas, and pagination logic.
  </Step>
  <Step title="Generate API client">
    A production-ready API client is generated with type hints, error handling, authentication support, and full documentation.
  </Step>
</Steps>

## Limitations

<Warning>
  Reverse API Engineer executes code locally using Claude Code. Monitor the generated output before running in production.
</Warning>

- Some websites use advanced bot-detection that may limit capture or require manual interaction
- The tool is designed for legitimate API research and development purposes
