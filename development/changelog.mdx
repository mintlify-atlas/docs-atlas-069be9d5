## v0.5.0 <span style={{color: '#6b7280', fontSize: '0.875rem', fontWeight: 'normal'}}>2026-03-17</span>

### Added

- **Follow-up chat** ‚Äî After a run completes, type follow-up messages to iterate in the same session without creating new run IDs or folders. Press Enter to finish.
- **Abort run (Ctrl+C)** ‚Äî Gracefully cancel a running agent or engineer session and return to the REPL instead of exiting the app.
- **Free-text prompts** ‚Äî All select/checkbox prompts now include an "Other (type your answer)" option. The agent prompt documents free-form, multi-select, and multi-question capabilities.
- **Random task suggestions (Ctrl+R)** ‚Äî Press Ctrl+R in agent mode to fill the prompt with a random curated task idea. Press again to cycle.

### Fixed

- **Usage tracking across follow-ups** ‚Äî Token counts now accumulate across all turns in a session instead of being overwritten by the last turn.
- **Agent mode follow-up** ‚Äî Auto engineer (agent mode) was using a copy-pasted streaming loop without follow-up support; now reuses the shared conversation loop.

---

## v0.4.5 <span style={{color: '#6b7280', fontSize: '0.875rem', fontWeight: 'normal'}}>2026-03-15</span>

### Fixed

- **Stream closed errors (#51)** ‚Äî Reverted from `query()` to `ClaudeSDKClient`, which maintains a persistent bidirectional connection, eliminating "Error in hook callback hook_0: Stream closed" errors.
- **CLAUDECODE env var leak** ‚Äî Clears the inherited `CLAUDECODE` env var from CLI subprocess to prevent nested session interference when running inside Claude Code.
- **CLI stderr noise** ‚Äî Minified JS stack traces are filtered to a single clean error line. Use `DEBUG=1` for full output.

### Changed

- **claude-agent-sdk** ‚Äî Bumped minimum version to 0.1.48.
- **Agent mode** ‚Äî No longer prompts for a URL; the agent navigates autonomously.
- **Header UI** ‚Äî Version and task labels now use mode-specific colors: agent = coral, engineer = blue, collector = gold.

## v0.4.4 <span style={{color: '#6b7280', fontSize: '0.875rem', fontWeight: 'normal'}}>2026-03-15</span>

### Fixed

- **Stream closed errors (#51)** ‚Äî Partial fix using `query()` with env var workarounds. Superseded by v0.4.5.

## v0.4.3 <span style={{color: '#6b7280', fontSize: '0.875rem', fontWeight: 'normal'}}>2026-03-12</span>

### Fixed

- **AskUserQuestion interactive prompt** ‚Äî Fixed interactive questionary UI not rendering in engineer and agent modes. Switched from `ClaudeSDKClient` to `query()` function, as `ClaudeSDKClient` silently auto-approves `AskUserQuestion` without triggering the `can_use_tool` callback.

---

## v0.4.2 <span style={{color: '#6b7280', fontSize: '0.875rem', fontWeight: 'normal'}}>2026-03-12</span>

### Fixed

- **Engineer client resolution** ‚Äî Prefer the active engineer client recorded in session history over config defaults, preventing iterative edits from switching to the wrong file when multiple language clients exist.

---

## v0.4.1 <span style={{color: '#6b7280', fontSize: '0.875rem', fontWeight: 'normal'}}>2026-03-12</span>

### Added

- **`show` CLI subcommand** ‚Äî View details of a specific run by ID.
- **Chrome Extension improvements**
  - Multi-session support with session isolation fixes
  - Codegen recording mode
  - Side panel UX overhaul: message persistence across reloads, traffic list with expandable request details, clear button
  - Restricted page detection with user feedback
  - ANSI escape code stripping in terminal output
  - PrismLight language fallback for unregistered languages
  - Bundle size reduced from 1112 KB to 436 KB (removed shiki, lucide-react, ansi-to-react, @base-ui/react)
- **Mintlify API client example**

### Changed

- **Background update check** ‚Äî Version check now runs in a background thread to avoid blocking CLI startup.

### Fixed

- **Engineer language preservation** ‚Äî Iterative edits now preserve the original output language instead of resetting.
- **Native host Gatekeeper handling** ‚Äî Scoped `xattr` to the actual claude-code package directory.
- **Codegen selector escaping** ‚Äî Dynamic attribute values now escaped with `CSS.escape()` to prevent broken selectors.
- **Chrome Extension manifest** ‚Äî Removed unused `scripting` permission; pinned devicon CDN to v2.16.0.

---

## v0.4.0 <span style={{color: '#6b7280', fontSize: '0.875rem', fontWeight: 'normal'}}>2026-03-10</span>

### Added

- **`list` CLI subcommand** ‚Äî Non-interactive command to list generated scripts and runs.
  - Rich table output (compact and full modes) and JSON output (`--json`)
  - Filter by mode (`--mode`), model (`--model`), prompt search (`--search`), and limit (`--limit`)
  - Shows script directory paths, file counts, and local path detection
